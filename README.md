# LAN文件服务器

一个基于Python的局域网文件共享服务器，支持多媒体文件浏览、搜索和移动端访问。

## 核心功能

- 🔐 **安全认证**：用户名密码验证、防暴力破解、智能会话管理
- 📁 **文件管理**：局域网内文件共享、多媒体文件预览、快速搜索
- 📱 **移动端优化**：响应式设计，适配手机和平板
- ⚙️ **配置灵活**：JSON配置文件，支持热重载
- 🔍 **高级搜索**：支持正则表达式和模糊匹配
- 📊 **增强日志**：彩色日志输出，便于问题追踪
- 🚀 **性能优化**：多级缓存、异步索引、sendfile优化

## 快速开始

### 环境要求
- Python 3.7+
- Windows/Linux/macOS

### 运行方式
```bash
# 直接运行
python server.py

# 使用PowerShell启动脚本（Windows）
.an-file-server-launcher.ps1
```
